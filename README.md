# Themed Tic Tac Toe

A fun, single-file browser tic-tac-toe game with switchable seasonal themes.

## Play

Open `tic-tac-toe.html` in any browser — no build step, no dependencies.

## Features

- **Two-player** tic-tac-toe with win/draw detection and a highlighted winning line.
- **5 themes**, switchable anytime (even mid-game):
  - 🎮 Classic — plain `X` / `O`
  - 🎄 Christmas — falling snow, 🎄 vs ⛄
  - 🧧 New Year — bursting fireworks, 🧧 vs 🐉
  - ❤️ Valentine — floating hearts, ❤️ vs 💌
  - 🎃 Halloween — drifting bats & ghosts, 🎃 vs 👻
- Animated full-screen background per theme.
- **New Game** keeps your chosen theme.

Everything lives in one self-contained HTML file (HTML + CSS + vanilla JS).
